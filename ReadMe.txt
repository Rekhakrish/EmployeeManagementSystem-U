PROJECT OVERVIEW
----------------------------------------------------
This project is a complete Employee Management System built using ASP.NET MVC (C#), ADO.NET with Entity Framework (EDMX), SQL Server, jQuery, and Bootstrap.

The application allows users to:
- Create, read, update, and delete employee records.
- Perform client-side validation using jQuery.
- Dynamically calculate age from Date of Birth.
- Handle pagination, sorting, searching, and batch deletion.
